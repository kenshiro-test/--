# Dream Schedule Planner ショート動画

YouTube Shorts / TikTok 向けの縦型解説動画です。

- サイズ: 1080 x 1920
- 長さ: 約22秒
- FPS: 24
- 形式: MP4

## 書き出しファイル

- `output/dream-schedule-planner-short.mp4`

## 構成

1. 予定作成の悩みを提示
2. ホームのカレンダーから日付を選ぶ
3. パークと見たいショーを選ぶ
4. 自動 / 手動で作成方法を選ぶ
5. ショーの時間を見ながら予定を調整
6. 画像保存・入園後やることメモまで紹介

## 再生成

この環境では `npm/npx` と `ffmpeg` がないため、実際のMP4生成は macOS の AVFoundation を使う Swift スクリプトで行っています。

Remotion で編集したい場合は、`remotion/` フォルダの構成を使って、Remotion が入った環境でレンダリングできます。
